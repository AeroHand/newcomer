# 入职培训

该仓库主要用于火盒公司新入职员工的培训课程，课程的目的是让员工快速融入到火盒公司的开发中，并通过课程
学习有效掌握其所属职位的工具，技能和知识点。

课程根据知识类别划分，每份课程都会配备相关的阅读，练习和思考题。团队负责人根据项目需求，指导新入职人员选择课程并在实际学习中给予一定帮助和指导。
